Assimp - RapidJSON

RapidJSON is a JSON parser and generator for C++. Note that the bin/jsonchecker/ directory mentioned in the license is not included.

Used in Qt 3D.

The sources can be found in qt-src/qt3d/src/3rdparty/assimp/contrib/openddlparser.
