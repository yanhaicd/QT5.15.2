Dear ImGui - stb

Single-file public domain (or MIT licensed) libraries for C/C++

Used in tests to demonstrate and verify the integration of an external GUI library.

The sources can be found in qt-src/qt3d/src/3rdparty/imgui/imstb_rectpack.h,imstb_textedit.h,imstb_truetype.h.
