Wagyu Geometry Processing Library, version 0.4.3

A library for basic geometric operations.

Used in the Mapbox GL plugin of Qt Location.

The sources can be found in qt-src/qtlocation/src/3rdparty/mapbox-gl-native/deps/wagyu.
